================
For Developers
================


If you want to expand or change an existing content element, take a look at the :ref:`templating <templating>` page.
For comprehensive developer information, such as which Schema.org Things are supported or which options are available for your own implementation, see the `developer documentation <https://docs.typo3.org/p/brotkrueml/schema/master/en-us/Developer/Index.html>`__ from schema.
Otherwise feel free to contact us via email typo3@diemedialen.de.
